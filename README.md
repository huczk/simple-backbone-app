# simple-backbone-app

How to install:
* download all files
* npm install (install npm modules)
* npm start (start json-server, http-server and automatically open the browser)

App has [json-server](https://github.com/typicode/json-server) database for testing and uses [http-server](https://github.com/indexzero/http-server) for serving files in the browser.
In the "DB SQL - project" folder are files with project of SQL database. 

Here is the link to apiary with api project: http://docs.booksapi19.apiary.io (remember to switch to mock server for testing).
